# DevPilot AI

## AI-Powered Production Workflow Assistant

DevPilot AI is an AI-native production orchestration platform that automates software workflow analysis, testing intelligence, deployment risk prediction, and CI/CD generation.

## Problem Statement

Traditional production workflows are manual, reactive, and slow. Developers spend significant time reviewing code, identifying deployment risks, generating test cases, and managing CI/CD pipelines.

## Solution

DevPilot AI uses Generative AI to automate critical stages of software production workflows.

## Features

* AI Code Review
* Bug Detection
* Security Analysis
* Automated Test Case Generation
* Deployment Risk Prediction
* CI/CD YAML Suggestions

## Tech Stack

* PHP
* HTML/CSS/JavaScript
* Gemini API

## Future Scope

* GitHub Integration
* AI Sprint Planning
* Automated Documentation
* Multi-Agent Workflow Automation
* Real-time Deployment Monitoring

## How to Run

1. Install XAMPP
2. Place project in htdocs folder
3. Start Apache
4. Open:
   http://localhost/devpilot-ai
